Risk prediction system for sprint delays in Scrum projects based on velocity statistical analysis

How to Use the DSS System

  1. Step 1: Fill in the "Input" tab

    Enter current sprint parameters: sprint duration, current day, planned Story Points, and completed Story Points.

  2. Step 2: Add historical data

    In the "History" tab, add velocity data from previous sprints. The more data, the more accurate the forecast.

  3. Step 3: View results

    The system automatically calculates forecasts, prediction intervals, and deviations from the plan.

  4. Step 4: Get recommendations

    Based on calculations, the system provides recommendations for risk management actions.
